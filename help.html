Panduan Penggunaan SISTA CSSD

Selamat datang di pusat dokumentasi resmi. Aplikasi SISTA-CSSD adalah sistem cerdas berbasis serverless yang mengintegrasikan pemindaian QR Code kamera, pelacakan siklus sterilisasi, gamifikasi staf, dan notifikasi otomatis multi-channel untuk mendukung akreditasi paripurna rumah sakit.

Masuk Aplikasi

1. Ekosistem Fitur Canggih

Mengapa SISTA-CSSD menjadi pelopor digitalisasi logistik steril rumah sakit.

Scan QR Kamera Cepat

Integrasi pustaka pemindaian kamera HP secara langsung. Cukup arahkan kamera pada label QR alat medis untuk melihat isi set, status sterilisasi, masa aktif, serta langsung memproses penyerahan/pengembalian instan.

Sistem Prestasi Staf 🏆

Menghadirkan fitur gamifikasi kerja. Staf CSSD mendapatkan akumulasi poin otomatis untuk setiap aksi sterilisasi, penyiapan alat, dan pencatatan. Dilengkapi halaman podium interaktif 3 besar teraktif bulanan.

Tracking 6-Siklus CSSD

Mengontrol status posisi instrumen medis secara komprehensif mulai dari Kotor, Pencucian, Proses Steril, Penyimpanan (di CSSD), Steril (Siap Pakai), hingga Dipinjam (di Ruangan).

Google Cloud Enterprise SSO

Login super aman menggunakan protokol otentikasi OAuth2 resmi Google. Backend memverifikasi token JWT secara berkala untuk menjaga kerahasiaan data pengguna dari pihak luar.

Notifikasi Multi-Channel

Setiap aksi penting (registrasi akun baru, persetujuan admin, pembuatan peminjaman, barang siap diambil, selesai) memicu email HTML otomatis dan notifikasi real-time via bot Telegram ke grup koordinasi.

Auto-Backup & Ekspor Cepat

Dukungan pustaka SheetJS untuk ekspor sekali klik berkas Excel (.xlsx) untuk monitoring data logistik ruangan. Super Admin juga dapat mengunduh seluruh database (Backup) dalam satu file ber-tab.

2. Siklus Pemrosesan Alat Medis

Gambaran siklus perputaran instrumen steril secara disiplin di rumah sakit.

1

Kotor

Alat pasca-tindakan medis dari ruangan dikembalikan ke CSSD dalam kondisi kotor.

2

Pencucian

Proses dekontaminasi, pencucian, pembilasan, pengeringan, dan pengecekan kelayakan fisik.

3

Proses Steril

Alat dikemas dan dimasukkan ke mesin autoklaf/suhu rendah untuk proses membunuh kuman.

4

Penyimpanan

Alat pasca-sterilisasi yang belum dilabeli/diverifikasi disimpan sementara di rak khusus CSSD.

5

Steril (Siap)

Alat diverifikasi, dilabeli barcode/QR, dan tampil di katalog ruangan sebagai barang siap dipinjam.

6

Dipinjam

Alat dibawa dan diserahterimakan ke ruangan/unit untuk digunakan kembali pada pasien tindakan.

3. Panduan Staf Ruangan / Unit Kerja

Petunjuk lengkap bagi unit peminjam alat sterilisasi medis.

Fase 1

Registrasi Cepat Berbasis Ruangan

Gunakan login akun Google Anda. Jika email Anda belum terdaftar di database, sistem secara otomatis mengalihkan Anda ke layar registrasi. Lengkapi nama Anda, pilih ruangan Anda (jika tidak tertera di pilihan utama, klik Lainnya dan ketik nama ruangan kustom Anda), serta isi nomor kontak HP/WhatsApp. Setelah mengklik kirim, notifikasi aktivasi otomatis terkirim ke Admin CSSD. Anda akan menerima email notifikasi setelah akun diaktifkan.

Fase 2

Mengajukan Peminjaman via Katalog Pintar

Pada halaman Katalog Alat, Anda dapat meninjau alat-alat yang saat ini berkondisi Steril di instalasi CSSD. Cari nama alat medis pada bilah pencarian, klik tombol + Keranjang. Buka tab Keranjang Request, tambahkan catatan opsional jika diperlukan (misalnya detail tindakan atau nama dokter penanggung jawab), lalu klik Ajukan Peminjaman. Setelah dikonfirmasi, sistem otomatis mengirimkan notifikasi ke pihak CSSD untuk segera mengemas barang.

Fase 3

Monitoring Status & Pemindaian QR Code Mandiri

Semua transaksi ruangan Anda dapat dipantau di tab Riwayat & Status. Anda dapat melacak status kesiapan alat (Pending, Ready, Aktif, Selesai). Selain itu, Anda dapat memindai label QR pada alat medis di ruangan Anda kapan saja. Arahkan kamera ponsel, sistem akan membuka pop-up Detail Hasil Scan yang berisi: ID Alat, Nama Alat, Komponen lengkap isi set alat tersebut, dan status posisinya saat ini untuk menjamin kesesuaian logistik steril secara akurat.

4. Panduan Petugas (Admin & Super Admin CSSD)

Tata cara pengelolaan transaksi logistik steril, penanganan ketidaklengkapan, dan gamifikasi poin.

Modul Admin

Pemrosesan Cepat Berbasis Scan (Zero-Typing Handover)

Petugas CSSD dapat mempercepat pemrosesan serah terima dan pengembalian alat medis dengan memindai kode QR fisik pada set alat medis menggunakan kamera:

  • Proses Serah Terima: Arahkan pemindai pada alat medis berstatus Ready for Pickup, modal hasil scan akan muncul. Klik Proses Serah Terima, Anda akan dialihkan secara instan ke halaman monitor dengan filter pesanan terkait, dan pop-up konfirmasi penerima otomatis terbuka.
  • Proses Pengembalian: Pindai alat berstatus Dipinjam, klik Proses Pengembalian, sistem akan mengarahkan ke menu monitor peminjaman dan membuka pop-up konfirmasi pengembalian alat medis.
Modul Admin

Manajemen Siklus Sterilisasi & Pelacakan Kadaluwarsa

Buka tab Inventaris Alat untuk melacak daur hidup alat. Ketika alat berstatus Kotor, klik ikon edit untuk mengubah siklus ke Pencucian, lalu Proses Steril. Setelah autoklaf selesai, set status alat ke Steril, sistem akan meminta Anda memasukkan tanggal sterilisasi dan secara otomatis mengkalkulasi tanggal kedaluwarsa berdasarkan masa aktif steril alat tersebut (misalnya 14, 30, atau 180 hari). Alat yang kedaluwarsa secara otomatis ditandai bahaya oleh sistem di database.

Modul Admin

Evaluasi Kinerja Staf & Akumulasi Poin Prestasi

Setiap petugas CSSD yang terdaftar di sistem akan mendapatkan poin otomatis setiap kali memproses transaksi logistik steril:

+10 Poin Menyelesaikan Sterilisasi Alat (Set Steril)
+5 Poin Melakukan Konfirmasi Serah Terima ke Ruangan
+5 Poin Memproses Konfirmasi Pengembalian Selesai

Perolehan poin bulanan dikompilasi secara real-time pada tab Prestasi Staf 🏆, lengkap dengan visualisasi podium untuk 3 peringkat teratas. Fitur ini dirancang untuk memacu dedikasi dan budaya pelayanan prima di internal CSSD.

5. Pertanyaan Umum (FAQ)

Jawaban cepat atas pertanyaan operasional yang paling sering diajukan.

Q: Bagaimana jika alat medis steril telah melewati tanggal kedaluwarsa steril?

Sistem secara otomatis mendeteksi tanggal kedaluwarsa steril. Jika masa aktif steril terlampaui, alat medis tersebut akan ditandai dengan warna merah bahaya dan otomatis disembunyikan dari katalog ruangan sehingga tidak bisa dipinjam. Alat medis fisik harus dikirim ulang ke CSSD untuk disterilkan kembali melalui siklus autoklaf.

Q: Bagaimana cara melaporkan jika ada alat yang hilang atau rusak saat dikembalikan?

Saat memproses pengembalian di tab Monitor Peminjaman, klik opsi Barang Tidak Lengkap / Rusak pada pop-up konfirmasi. Masukkan deskripsi kehilangan/kerusakan fisik alat, ambil foto bukti fisik menggunakan kamera ponsel, lalu simpan. Sistem akan menandai transaksi tersebut dengan status Tidak Lengkap (merah bahaya) dan mengirim notifikasi WhatsApp/Telegram kepada perwakilan ruangan agar segera ditindaklanjuti.

Q: Apakah aplikasi SISTA-CSSD dapat diinstal di ponsel (HP) tanpa masuk ke Play Store?

Ya! Aplikasi SISTA-CSSD mendukung teknologi PWA. Buka tautan aplikasi di browser Google Chrome (Android) atau Safari (iOS) ponsel Anda, tekan menu browser, lalu pilih Tambahkan ke Layar Utama / Install App. Aplikasi akan terinstal langsung di layar ponsel Anda dan dapat diakses dengan cepat seperti aplikasi asli.

Q: Bagaimana jika akun pendaftaran staf tertunda (Pending)?

Demi menjaga keamanan logistik instrumen bedah rumah sakit, setiap akun pendaftar ruangan baru wajib diverifikasi oleh Kepala Unit CSSD. Silakan hubungi admin CSSD melalui tombol pintasan Hubungi Admin di menu sidebar untuk aktivasi cepat akun Anda.
